# Casa da Itália

Este é um projeto de portfólio simples para o site da Casa da Itália, um restaurante italiano fictício. O site apresenta informações básicas sobre o restaurante, incluindo menu, localização e contato.

## Tecnologias Utilizadas

- **HTML**: Linguagem de marcação utilizada para estruturar o conteúdo da página web.
- **CSS**: Linguagem de estilo utilizada para definir a aparência visual do site, incluindo layout, cores e tipografia.

## Como Executar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador web moderno. Não há necessidade de servidor ou dependências adicionais, pois é um site estático.

1. Clone ou baixe este repositório.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` no seu navegador.

## Estrutura do Projeto

- `index.html`: Arquivo principal contendo a estrutura HTML da página inicial.
- `Visual/style.css`: Arquivo de estilos CSS para personalizar a aparência do site.
